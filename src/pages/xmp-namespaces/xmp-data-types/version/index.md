# Version

Describes one version of a document.

- The namespace URI is [http://ns.adobe.com/xap/1.0/sType/Version#](http://ns.adobe.com/xap/1.0/sType/Version#)

- The preferred namespace prefix is stVer

|    |           |    |
|----|-----------|----|
|**Name**|**Description**|**Type**|
|stVer:comments|Comments concerning what was changed  |[Text](../../../xmp-namespaces/xmp-data-types/index.md#text)|
|stVer:event|High-level, formal description of what operation the user performed  |[ResourceEvent](../resource-event/index.md)|
|stVer:modifier|The person who modified this version  |[ProperName](../../../xmp-namespaces/xmp-data-types/index.md#propername)|
|stVer:modifyDate|The date on which this version was checked in  |[Date](../../../xmp-namespaces/xmp-data-types/index.md#date)|
|stVer:version|The new version number  |[Text](../../../xmp-namespaces/xmp-data-types/index.md#text)|
